273  // matrix - vector multiplication
274  // although opengl uses premultiplication
275  // because of the way matrices are used in opengl:
276  //
277  // ofMatrix4x4 openGL
278  // [0] [1] [2] [3] [0] [4] [8] [12]
279  // [4] [5] [6] [7] [1] [5] [9] [13]
280  // [8] [9] [10] [11] [2] [6] [10] [14]
281  // [12] [13] [14] [15] [3] [7] [11] [15]
282  //
283  // in memory though both are layed in the same way
284  // so when uploading a matrix it just works without
285  // needing to transpose
286  //
287  // so although opengl docs explain transformations
288  // like:
289  //
290  // ofVec3f c = rotateZ 30 ofVec3f a around ofVec3f b
291  //
292  // openGL docs says: c = T(b)*R(30)*a;
293  //
294  // with ofMatrix4x4:
295  // ofMatrix4x4 R = ofMatrix4x4::newRotationMatrix(30,0,0,1);
