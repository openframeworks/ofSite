##InlineDescription

Allows to iterate over a string's utf8 codepoints.
The easiest way to use it is with a c++11 range style
for loop like:

for(auto c: ofUTF8Iterator(str)){
...
}

which will iterate through all the utf8 codepoints in the
string.
